# PersonalPortfolio

My personal website, built with Blazor (.NET 10). It holds my intro, skills,
work experience, and projects, and I keep it updated over time.

## Run locally

```bash
dotnet run --project src/PersonalPortfolio.Web
```

Then open the URL printed in the console.

## Structure

```
src/PersonalPortfolio.Web   Blazor Web App
docs                        Notes
```

Most of the content lives in `src/PersonalPortfolio.Web/Models/Profile.cs`.
